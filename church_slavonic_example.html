Font Recommendations

For best results with Church Slavonic text, consider using these fonts:

  • Ponomar - Specifically designed for Church Slavonic
  • serif - Fallback with good Unicode support

CSS Usage

To use the Church Slavonic styling in your Markdown files, include the CSS file:

<link rel="stylesheet" href="church_slavonic.css">

Or if you're using a Markdown processor that supports CSS, you can include the styles directly in your document.
